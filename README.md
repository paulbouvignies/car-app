## Installing / Getting started
_____
### Front: 
```bash
cd vitrine
npm install
npm run dev
```
Now you can access the front on http://localhost:3000

### Storybook
```bash
cd design-system
npm install
npm run storybook
```
Now you can access the storybook on http://localhost:6006
